About This Project

Friendbot is an automated self-help for users in the guise of a chatbot. This project is a comprehensive study on years of research done in the field of Therapy Artificial Intelligence and an application based on the same. Using Chatterbot as the main library, a self-created dataset has been used to train and test to enable a comprehensible conversational flow. A further goal is to work on optimization algorithms to obtain better-enhanced dialogues.

Algorithms

A comprehensive decription of algorithms that measure text similarity

Levenshtein Distance

Minimum number of single-character edits required to change one word into the other.

Jaccard Similarity

Compares members for two sets to see which members are shared and which are distinct

Synset Distance

Different set of words grouped as a synonymy set.

Sentiment Comparison

Sentiment comparison algorithms calculate the closeness of two statements based on the closeness of the sentiment value of the two statements.
